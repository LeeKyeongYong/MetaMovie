아빠의 희귀한 공룡 화석을 자랑하는 비실이에게 발끈해 공룡알 화석을 찾아내겠다고 선언한 진구.무작정 뒷산에서 공룡알을 찾던 진구는 우연히 알 모양의 화석을 발견하게 되고, 도라에몽의 타임보자기를 이용해 부화에 성공한다. 진구는 알에서 깨어난 아기 공룡에게 '피스케'라는 이름을 지어주고 집안에서 엄마 몰래 키우기 시작한다.하지만 무럭무럭 자라는 피스케를 더 이상 감추지 못하게 되자 마을 연못에 숨겨두는 진구. 하지만 감기에 걸린 진구가 걱정된 피스케가 어마어마하게 자라난 몸을 이끌고 마을에 나타나면서 큰 소동이 벌어진다.진구는 어쩔 수 없이 피스케를 공룡들이 살던 시대로 돌려 보내기로 결심하고, 진구와 도라에몽 그리고 친구들은 피스케를 위해 타임머신을 타고 1억년 전 백악기 공룡시대로 떠나게 되는데..
